# Usage of fluid system

```
KEY_W # viewpoint moves forward
KEY_A # viewpoint moves left
KEY_S # viewpoint moves backward
KEY_D # viewpoint moves right
KEY_R # reset fluid
KEY_B # fluid simulation with surface
KEY_V # fluid simulation without surface
KEY_ESC # exit
MOUSE WHEEL # scaling all
```
